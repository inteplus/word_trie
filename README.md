# wordtrie

A small library for constructing a word trie and using the word trie to estimate probabilities.

## Install

### C++

Use CMake::

```
mkdir cpp_build
cd cpp_build
cmake ..
make
```


### Python

Use setuptools::

```
python3 setup.py {bdist_wheel|develop|install}
```
